# GIỚI THIỆU DỰ ÁN GIFT SHOP (GIS)
Dự án quản lý cửa hàng quà tặng tích hợp bản đồ.

## Thành viên thực hiện
- **Hoàng Nhật**: Quản lý chung & Cấu hình hệ thống (Nhánh `dev`)
- **TanPhat**: Module Quản lý tài khoản (Nhánh `dev-accounts`)
- **ThanhPhat**: Module Sản phẩm & Giỏ hàng (Nhánh `dev-products`)

## Công nghệ sử dụng
- Python / Django
- PostgreSQL / PostGIS (GIS)
- HTML/CSS/JS (Frontend)
